Installation

Comment installer shadcn-docs-nuxt dans votre application.

Utilisation du template de démarrage

Obtenir le Template de Démarrage

npm
pnpm
bun
yarn
npx nuxi@latest init <project-name> -t github:ZTL-UwU/shadcn-docs-nuxt-starter

pnpm dlx nuxi@latest init <project-name> -t github:ZTL-UwU/shadcn-docs-nuxt-starter

bun x nuxi@latest init <project-name> -t github:ZTL-UwU/shadcn-docs-nuxt-starter

yarn dlx nuxi@latest init <project-name> -t github:ZTL-UwU/shadcn-docs-nuxt-starter

Alternativement, vous pouvez cloner ou télécharger le template depuis le dépôt GitHub.

Navigation vers le Répertoire du Projet

cd <project-name>

Serveur de Développement

npm
pnpm
bun
yarn
npm run dev

pnpm run dev

bun run dev

yarn run dev

Parfait ! Une fenêtre de navigateur devrait s'ouvrir automatiquement sur http://localhost:3000.
