ប្លុកសម្រាប់សរសេរកូដ

ការបង្ហាញជម្រើសផ្សេងៗនៃប្លុកសម្រាប់សរសេរកូដ។

ការប្រើប្រាស់

export default () => {
  console.log('Code block');
};

```ts
export default () => {
  console.log('Code block');
};
```

ជាមួយចំណងជើង

nuxt.config.ts
export default defineNuxtConfig({
  // My Nuxt config
});

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  // My Nuxt config
});
```

Highlight បន្ទាត់

console.log('1');
console.log('2');
console.log('3');
console.log('4');

```ts {1,3-4}
console.log('1')
console.log('2')
console.log('3')
console.log('4')
```

```ts{1} line-numbers

```

ការវិវត្តិប្រែប្រួលកូដ

ដកសញ្ញា \ ពីក្នុង notions ចេញដើម្បីប្រើប្រាស់ការវិវត្តិប្រែប្រួលកូដ

e.g. ដូរ // [\!code --] ទៅកាន់ // [!code --]

ភាពខុសគ្នា

console.log('hewwo'); console.log('hello'); console.log('goodbye');

```ts
console.log('hewwo'); // [\!code --]
console.log('hello'); // [\!code ++]
console.log('goodbye');
```

ការ Error

console.log('No errors or warnings');
console.error('Error'); console.warn('Warning'); 
```ts
console.log('No errors or warnings');
console.error('Error'); // [\!code error]
console.warn('Warning'); // [\!code warning]
```

ផ្តោត

console.log('Not focused');
console.log('Focused'); console.log('Not focused');

```ts
console.log('Not focused');
console.log('Focused'); // [\!code focus]
console.log('Not focused');
```
