ការណែនាំ

shadcn-docs-nuxt ជាគម្រូសំណេរឯកសារសម្រាប់ Nuxt ដោយប្រើប្រាស់ Nuxt Content និង shadcn-vue។

ការលើកទឹកចិត្ត

shadcn-docs-nuxt ត្រូវបានបង្កើតឡើងមកក្នុងគោលបំណងជំនួសឲ្យការប្រើប្រាស់ Docus និង Nuxt UI Pro Docs។

មុខងារ

  • ឥតគិតថ្លៃ និង បើកប្រភពកូដ។
  • អាច កែច្នៃតាមចំណង់ចំណូលចិត្ត។
  • សម្បូរទៅដោយ components ជាច្រើន។
  • គាំទ្រលើការបង្ហាញទូរស័ព្ទ។
  • ងាយស្រួលត្រូវបានស្វែងរកដោយ Search Engine ដែលមានជំនួយពី Nuxt Content។
  • មានបន្ថែម component ដែលមកពី Docus, Nuxt UI Pro Docs, undocs។

ក្រេឌីត

  • Nuxt Content: ងាយស្រួលបង្កើតមាតិកាសម្រាប់ Vue Developers។
  • shadcn-ui: សម្រាប់ component ដែលស្រស់ស្អាត និងសម្បូរបែប។
  • shadcn-vue: ជាការនាំចូល component សម្រាប់ Vue ពី shadcn-ui និងប្រភពផ្សេងៗទៀត។
  • Docus: យកជាគម្រូ និងការរចនា component មួយចំនួន។
  • Nuxt UI Pro Docs: យកជាគម្រូ។
